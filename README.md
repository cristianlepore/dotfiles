# Dotfiles

Questo repository contiene i file di configurazione (dotfiles) per un ambiente Linux, ottimizzati per Bash, Zsh e vari editor di testo.

## Contenuto della cartella

- **`.bash_logout`**: Gestisce le operazioni di uscita da Bash (pulisce lo schermo).
- **`.gitconfig`**: Configurazioni globali di Git, inclusi i dati dell'utente e il supporto per Git LFS.
- **`.p10k.zsh`**: Configurazione dettagliata per il tema Powerlevel10k di Zsh, definendo lo stile del prompt.
- **`.profile`**: Configurazione per le shell di login, gestisce il `PATH` e carica `.bashrc`.
- **`.tmux.conf`**: File di configurazione per il multiplexer di terminali tmux.
- **`.vimrc`**: Personalizzazione dell'editor Vim (sintassi, plugin, ecc.).
- **`.zshrc`**: Configurazione principale della shell Zsh, integra Oh My Zsh e il tema Powerlevel10k.

## Utilizzo

Questi file sono pensati per essere linkati o copiati nella propria home directory per personalizzare l'ambiente di lavoro.
